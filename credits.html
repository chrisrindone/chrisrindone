For whatever crazy reason, (peole who know me IRL would understand), I can't settle on just one editor. As such, I shuffle between VS Code and Brackets.

There are numerous online tools that really help me to speed up my workflow. Part of why is because my memory isn't what it once was, another part is a convenience, and the last part is laziness :-)! Here are some of my go to's:

  • HSL Color Picker - Awesome color tool which allows you to get your color codes in hex, rgba and hsla.
  • 30 Seconds of CSS - Self-described as, "A curated collection of useful CSS snippets you can understand in 30 seconds or less." Useful AND fun stuff!
  • Clippy - A CSS clip-path maker. Does what it says, and does so very intuatively.
  • Grid by Example - A great resource for getting started with CSS Grid from Rachel Andrew, whose depth and breadth of font-end development is nothing short of amazing!
  • LunaPic - An excellent, free, online photo editor.
  • Mozilla Developer Network (MDN) - Because knowledge.
